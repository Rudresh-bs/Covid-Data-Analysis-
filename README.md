🎯 **Objective** Analyze and visualize global COVID-19 impact across 243 countries from 2020 to 2023.

🔧 **Tools Used**

- Python (Pandas, Matplotlib, Seaborn)
- Google Colab
- Tableau Public

📁 **Dataset Source:** Our World in Data Records: 333,420 rows | 22 columns

📊 **Key Insights**

- USA recorded highest total cases globally
- Peak cases recorded in January 2022
- Europe had highest fatality rates
- Oceania & North America led vaccination rates
